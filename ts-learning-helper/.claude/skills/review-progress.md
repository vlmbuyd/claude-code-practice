# Review Progress Skill

현재 학습 상태를 분석하고 다음 단계를 구체적으로 추천합니다.

## 사용법

```
/review-progress
```

## 진행 순서

1. **진도 조회**: `npm run dev -- progress` 를 실행하세요.

2. **인사이트 조회**: `npm run dev -- insight list` 를 실행하세요.

3. **패턴 분석**: 다음을 분석하세요:
   - 인사이트가 적은 챕터는 어디인가? (이해가 얕을 수 있음)
   - 최근 2주간 어떤 챕터를 공부했는가?
   - 어떤 유형의 인사이트가 많은가? (개념? 패턴? 실수?)

4. **메타인지 지원**: 학습자에게 현재 상태를 가시적으로 보여주세요:
   - "강점 영역: ..." 
   - "보완이 필요한 영역: ..."
   - "학습 패턴 관찰: ..."

5. **다음 학습 추천**: 반드시 **하나만** 구체적으로 추천하세요:
   "다음으로 [챕터 N - 주제]를 공부하시길 추천합니다. 이유: ..."

## 주의사항

학습자는 메타인지가 부족한 편입니다. 현재 상태를 명확하고 구체적으로 보여주되,
압도되지 않도록 다음 단계는 하나만 제시하세요.
