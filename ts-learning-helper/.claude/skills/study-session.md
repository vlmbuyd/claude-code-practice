# Study Session Skill

이펙티브 타입스크립트의 특정 챕터를 체계적으로 학습하는 세션을 진행합니다.

## 사용법

```
/study-session
```

## 진행 순서

1. **챕터 확인**: "오늘 어느 챕터를 공부할까요?" 라고 물어보세요.

2. **키워드 단계**: `npm run dev -- keywords <챕터>` 실행 후 키워드를 함께 검토하세요. 학습자에게 이미 아는 키워드와 모르는 키워드를 구분하게 하세요.

3. **개념 탐색**: 모르는 키워드 중 가장 중요한 것을 골라 `npm run dev -- explain --feynman "<개념>"` 을 실행하세요.

4. **AI Free 원칙 확인**: 파인만 기법 전, "먼저 이 개념을 스스로 설명해볼 수 있나요?" 라고 물어보세요.

5. **연습 문제**: 개념 학습 후 관련 연습 문제를 직접 풀어보도록 제안하세요. `npm run dev -- solve "<문제>"` 를 활용하세요.

6. **인사이트 저장**: 세션 마무리 시 "오늘 학습에서 가장 중요한 인사이트 하나를 기록해봅시다." 라고 안내하고 `npm run dev -- insight add` 를 실행하세요.

7. **진도 기록**: `npm run dev -- progress mark <챕터>` 로 완료를 표시하세요.

## AI Free 원칙

매 단계에서 AI 도움을 주기 전에 반드시 확인하세요:
"먼저 스스로 해보시겠습니까?"

학습자가 포기하지 않도록 격려하되, 본인이 시도한 후에 도움을 제공하세요.
